## Installation

Comier is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'Comier'
```

### Using Carthage
Add to Cartfile
```
github 'baveku/Comier' 'carthage'
```
